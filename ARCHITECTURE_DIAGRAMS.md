# Визуальная архитектура логики добавления товара в корзину

## Диаграмма потока (Flow Diagram)

```
┌─────────────────────────────────────────────────────────────────┐
│                      КЛИЕНТ (JavaScript)                         │
│  1. Пользователь кликает "Добавить в корзину"                  │
│  2. Собирает данные: variant_id, quantity, CSRF token          │
│  3. POST запрос на /cart/add/                                   │
└────────────────────┬────────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────────┐
│              DJANGO VIEW (views_cart.py)                         │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 1. Проверка CSRF token                                  │   │
│  │    ✅ token есть в заголовке X-CSRFToken              │   │
│  │    ❌ token отсутствует → 403 Forbidden               │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 2. Валидация входных данных (CartValidator)           │   │
│  │    • validate_variant_id(variant_id)                   │   │
│  │      - Проверка: не пустой, целое число, > 0         │   │
│  │      ❌ Ошибка → ProductVariantNotFoundError (404)   │   │
│  │    • validate_quantity(quantity)                       │   │
│  │      - Проверка: целое число, от 1 до 999            │   │
│  │      ❌ Ошибка → InvalidQuantityError (400)           │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 3. Вызов CartService.add_item()                        │   │
│  │    → перейти в SERVICE слой                            │   │
│  └──────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────────┐
│              SERVICE СЛОЙ (cart_service.py)                      │
│                                                                   │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ @transaction.atomic  ← НАЧАЛО ТРАНЗАКЦИИ              │   │
│  │ def add_item(request, variant_id, quantity):           │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 1. Получить вариант товара с блокировкой (LOCK)       │   │
│  │    ProductVariant.objects.select_for_update().get()    │   │
│  │                                                         │   │
│  │    ⏳ БД блокирует эту строку                          │   │
│  │    ⏳ Другие процессы ждут                             │   │
│  │    ❌ Товар не существует → ProductVariantNotFound   │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 2. Проверить наличие товара на складе                  │   │
│  │    if product_variant.quantity < quantity:             │   │
│  │        raise InsufficientStockError(...)               │   │
│  │                                                         │   │
│  │    Вариант 1: Товара достаточно                        │   │
│  │      ✅ Продолжить                                     │   │
│  │                                                         │   │
│  │    Вариант 2: Товара недостаточно                      │   │
│  │      ❌ InsufficientStockError → 400                  │   │
│  │      ⏸  Откат транзакции, блокировка снята           │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 3. Получить или создать корзину                        │   │
│  │    cart = CartService.get_or_create_cart(request)      │   │
│  │                                                         │   │
│  │    Если пользователь авторизован:                       │   │
│  │      • Получить корзину по user_id                     │   │
│  │      • Если есть сессионная корзина:                   │   │
│  │        - Объединить товары (if not first time)         │   │
│  │        - Удалить сессионную корзину                    │   │
│  │                                                         │   │
│  │    Если пользователь анонимный:                         │   │
│  │      • Создать сессию если нет                          │   │
│  │      • Получить корзину по session_key                 │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 4. Получить или создать элемент корзины                │   │
│  │    CartItem.objects.get_or_create(                      │   │
│  │        cart=cart,                                       │   │
│  │        product_variant=variant,                         │   │
│  │        defaults={'quantity': quantity}                  │   │
│  │    )                                                    │   │
│  │                                                         │   │
│  │    Сценарий 1: Товара нет в корзине (created=True)    │   │
│  │      ✅ Создан CartItem с quantity=2                  │   │
│  │                                                         │   │
│  │    Сценарий 2: Товар уже в корзине (created=False)    │   │
│  │      • Проверить новое количество                      │   │
│  │        new_qty = old_qty + new_qty                     │   │
│  │      • Если нового количества > на складе:             │   │
│  │        ❌ InsufficientStockError → 400                │   │
│  │        ⏸  Откат транзакции, блокировка снята         │   │
│  │      ✅ Обновить CartItem.quantity = new_qty         │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 5. Логирование успешной операции                       │   │
│  │    logger.info(f"Cart item created: {item.id}")        │   │
│  └──────────────────────────────────────────────────────────┘   │
│                     │                                             │
│                     ▼                                             │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ @transaction.atomic  ← КОНЕЦ ТРАНЗАКЦИИ               │   │
│  │ return cart_item                                        │   │
│  │                                                         │   │
│  │ ✅ COMMIT к БД                                        │   │
│  │ ⏸  Блокировка освобождена                              │   │
│  └──────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────────┐
│         VIEW слой (exceptions handling)                          │
│                                                                   │
│  try:                                                            │
│      cart_item = CartService.add_item(...)  ← может выбросить  │
│  except InsufficientStockError as e:                           │
│      return JsonResponse({...}, status=400)                    │
│  except ProductVariantNotFoundError as e:                      │
│      return JsonResponse({...}, status=404)                    │
│  except CartOperationError as e:                               │
│      return JsonResponse({...}, status=500)                    │
│  except Exception as e:                                         │
│      logger.error(...)                                         │
│      return JsonResponse({...}, status=500)                    │
│                                                                   │
│  return JsonResponse({                                          │
│      'success': True,                                           │
│      'message': 'Товар добавлен в корзину',                   │
│      'cart_total': cart.total_items                           │
│  })                                                             │
└─────────────────────────────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────────┐
│              КЛИЕНТ (JavaScript обработка)                       │
│                                                                   │
│  response.status = 200  ✅                                      │
│    ↓                                                             │
│    • Показать сообщение об успехе                             │
│    • Обновить счётчик корзины                                  │
│    • Очистить форму                                             │
│                                                                   │
│  response.status = 400  ❌                                      │
│    ↓                                                             │
│    • Показать ошибку валидации                                 │
│    • Предложить исправить (количество, выбрать товар)          │
│                                                                   │
│  response.status = 404  ❌                                      │
│    ↓                                                             │
│    • Товар больше не доступен                                  │
│    • Предложить другой товар                                    │
│                                                                   │
│  response.status = 500  ❌                                      │
│    ↓                                                             │
│    • Техническая ошибка сервера                                │
│    • Попросить повторить позже                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Race Condition Prevention

```
                    ПРОЦЕСС 1              ПРОЦЕСС 2
                        │                      │
                        │                      │
        ┌───────────────▼──────────────────────▼─────────────┐
        │  variant.quantity = 5                             │
        │  (5 товаров на складе)                             │
        └────────────────────────────────────────────────────┘
                        │                      │
        ┌───────────────▼──────────────────────▼─────────────┐
        │  SELECT variant FOR UPDATE           │             │
        │  (LOCK acquired)                     │             │
        │  ✅ variant.id = 1 LOCKED            ⏳ waiting...  │
        └────────────────────────────────────────────────────┘
                        │                      │
        ┌───────────────▼──────────────────────┼─────────────┐
        │  if qty (5) < request (3):           │             │
        │    ❌ raise InsufficientStockError   │             │
        │  ✅ continue                         │             │
        └────────────────────────────────────────────────────┘
                        │                      │
        ┌───────────────▼──────────────────────┼─────────────┐
        │  CartItem.get_or_create(             │             │
        │    quantity = 3                      │             │
        │  )                                   │             │
        └────────────────────────────────────────────────────┘
                        │                      │
        ┌───────────────▼──────────────────────┼─────────────┐
        │  COMMIT TRANSACTION                  │             │
        │  (LOCK released)                     │             │
        │  ✅ cart has 3 items                 ▼ LOCK acquired│
        └────────────────────────────────────────────────────┘
                        │                      │
                        │        ┌─────────────▼────────────┐
                        │        │ if qty (5-3=2) < req(4):│
                        │        │   ❌ InsufficientStock   │
                        │        │ ✅ continue              │
                        │        └────────────────────────┬─┘
                        │                      │          │
                        │        ┌─────────────▼──────────┘
                        │        │
                        │        ▼
                        │    ┌──────────────┐
                        │    │ CartItem     │
                        │    │ quantity = 2 │
                        │    └──────────────┘
                        │
                    ✅ SUCCESS        ❌ FAIL (insufficient stock)
                    qty = 3           qty remains 3 (P2 failed)

Результат: total = 3 items (не 7! Предотвращена overselling)
```

## Последовательность операций при CSRF атаке

```
┌─────────────────────────────────────┐
│   ЗЛОУМЫШЛЕННИК                     │
│   Создаёт поддельную форму          │
│   <form action="...cart/add/">      │
│   (без CSRF token)                  │
└────────────┬────────────────────────┘
             │
             │ Убеждает пользователя
             │ кликнуть кнопку
             │
             ▼
┌─────────────────────────────────────┐
│   ПОЛЬЗОВАТЕЛЬ                      │
│   Кликает кнопку в браузере         │
│   Form data отправляется            │
│   (без CSRF token)                  │
└────────────┬────────────────────────┘
             │
             ▼
┌─────────────────────────────────────┐
│   DJANGO MIDDLEWARE                 │
│   CsrfViewMiddleware.process_request│
│   Проверить CSRF token              │
│   ❌ Token отсутствует              │
│   ❌ Tokens don't match             │
└────────────┬────────────────────────┘
             │
             ▼
┌─────────────────────────────────────┐
│   ОТВЕТ БРАУЗЕРУ                    │
│   HTTP 403 Forbidden                │
│   CSRF verification failed          │
└─────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────┐
│   ТОВАР НЕ ДОБАВЛЕН                 │
│   Корзина не изменилась             │
│   Пользователь защищён! ✅          │
└─────────────────────────────────────┘
```
