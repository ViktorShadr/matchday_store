Условия возврата

На этой странице размещается порядок возврата товаров, условия обращения по возврату и общие правила взаимодействия по таким запросам.

На главную В каталог

1. Общие положения

Настоящие Условия возврата определяют порядок обращения пользователя по вопросам возврата товаров, приобретённых через интернет-магазин атрибутики.

2. Основания для возврата

Возврат может осуществляться в случаях, предусмотренных законодательством и внутренними правилами магазина, включая:

  • обнаружение недостатков товара;
  • получение товара, не соответствующего заказу;
  • иные случаи, прямо допускаемые применимыми нормами.

3. Порядок обращения

Для рассмотрения вопроса о возврате пользователю рекомендуется обратиться в магазин с указанием данных заказа, описанием ситуации и контактной информации для обратной связи.

4. Проверка обращения

Каждое обращение рассматривается индивидуально. При необходимости могут быть запрошены дополнительные сведения, подтверждающие обстоятельства покупки и возврата.

5. Сроки и условия

Сроки, порядок и иные условия возврата определяются в соответствии с действующим законодательством, характером товара и обстоятельствами обращения.

6. Заключительные положения

Актуальная редакция условий возврата размещается на данной странице. При изменении подходов к обработке обращений информация может быть обновлена.
